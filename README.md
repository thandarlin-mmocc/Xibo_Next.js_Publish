# xibo-content-uploader-poc

## 概要（Overview）

本リポジトリは、**Xibo CMS v4 に画像やメディアをアップロードする Web システム**
の概念実証（PoC）を目的としたプロジェクトです。

一般ユーザーが Web 画面から画像を投稿し、その画像を **Xibo API** を通じて
Xibo CMS のメディアライブラリへ登録・表示できるかを検証します。

公共施設、図書館、地域コミュニティ向けの
デジタルサイネージ活用を想定しています。

---

## 目的（Objectives）

- Web ブラウザからの画像アップロードの検証
- Xibo CMS v4 の API 連携の実現性確認
- アップロードした画像の自動メディア登録
- 将来的な本番システム構築に向けた技術検証

---

## PoC スコープ

### 対象範囲（In Scope）

- 画像アップロード用の Web UI
- 画像形式：JPEG / PNG
- 基本的な入力項目（タイトル、名前）
- Xibo API を利用したメディアライブラリ登録
- 既存レイアウト・プレイリストへの手動割り当て（任意）

### 対象外（Out of Scope）

- ユーザー認証・認可
- 表示スケジュール（日付・時間制御）
- 画像内容の自動チェック・モデレーション
- 操作ログ・監査ログ
- 組織・テナント分離機能

> ⚠️ 本プロジェクトは **PoC（概念実証）専用** であり、  
> そのまま本番利用することは想定していません。
